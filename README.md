# gundam-backgrounds
Bunch of backgrounds for vscode

### Settings in `settings.json`

```
"background.enabled": true,
  "background.useDefault": false,
  "background.style": {
    "opacity": 0.15,
    "height": "100%",
    "size": "cover"
  },
 "background.customImages": [
    "https://raw.githubusercontent.com/HarisSpahija/gundam-backgrounds/main/RX.jpeg",
    "https://raw.githubusercontent.com/HarisSpahija/gundam-backgrounds/main/exia.jpeg",
    "https://raw.githubusercontent.com/HarisSpahija/gundam-backgrounds/main/strike.jpeg",
    "https://raw.githubusercontent.com/HarisSpahija/gundam-backgrounds/main/unicorn.jpeg",
  ]
```
